# File_Image_storage

## 项目内容
 

 - 一款可以存储照片、 备忘录或者动态的应用。 
 - 利用的是 IPFS 的分布式文件系统来弥补
以太坊存储问题，
 - 同时利用以太坊的非对称加密来解决 IPFS 文件存储的安全问

## 使用方式

 - 在本地打开Ganache
 - 然后进入文件目录之下，运行指令 npm start
 - 如果对 js 等文件进行改动，则需要使用 npm run dev-front
